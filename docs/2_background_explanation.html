ビジター招待アシスタント

この企画が生まれた背景
現在の課題：「招待案内のブレ」
毎週、複数のメンバーが独立した方法でビジター様を招待しています。
現状の問題：
• メンバー A：長めのメール文で招待
• メンバー B：LINE で短く招待
• メンバー C：電話で口頭説明のみ
• メンバー D：独自フォーマットで招待

↓

結果：招待内容にバラつきが生まれ、ビジター様が不安になる可能性があります
私たちが目指すこと：「ブレない招待」
改善後：
メンバー全員が「同じ形式」で「統一された案内」を送信

↓

ビジター様が安心して参加できる環境を作る
ビジター招待アシスタント とは？
MVP（Minimum Viable Product） = 「最小限の実行可能な製品」
つまり：
  • シンプル：3 つの項目を入力するだけ
  • 統一：誰が使ってもビジター様に同じ案内が届く
  • 効率的：手作業を減らし、自動生成される案内
✨ このアプリでできること
①入力が簡単
メンバーが入力するのはたった 3 つ：
• ビジター様のお名前
• 参加予定日
• 業種
②案内が自動生成される
📝
テキスト
丁寧で統一された案内文章がコピー可能に → LINE / メッセンジャーに貼り付け
📄
当日ガイド HTML
スケジュール、Zoom 情報、プレゼン構成案が全て入ったガイド → メール / LINE で送信可能に
📋
メンバー表 PDF
ビジター様が他のメンバーを事前に知ることができる
③Claude AI がプレゼン案を提案する
業種を選ぶと、AI が「こんなプレゼン構成はいかがですか？」と業種別のプレゼン案を自動提案します。
例）飲食業の場合：

①冒頭（5秒）：自社の飲食サービスの概要
②メイン（20秒）：提供する価値と特徴
③クロージング（5秒）：興味・問い合わせのお願い
📊 期待される効果
メンバー側
✅ 準備時間が大幅削減

ビジター様への案内準備が 30 秒で完了

✅ 迷わない

毎回「同じ形式」だから、何をしたらいいか明確

✅ 複雑な連絡が不要

手書きや複数の連絡先からの案内が不要に

ビジター様側
✅ 安心できる

統一された案内で、チャプターへの信頼が深まる

✅ スケジュール把握が完璧

当日のスケジュール全体が一目瞭然

✅ 準備しやすい

プレゼン構成案をもらえるので、準備が楽

ビジターホストチーム側
✅ 招待案内のバラつきが解消

メンバー間の「招待の質」が統一される

✅ ビジター様からの質問が減少

「スケジュールは？」「Zoom リンクは？」といった質問が激減

✅ 本当に大切な対応に集中

招待の「案内」に手間が取られず、「ビジター対応」に集中できる
