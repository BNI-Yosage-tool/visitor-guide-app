🚀 ビジター招待アシスタント

メンバーの皆さんへ、プロジェクト完成のお知らせ

お疲れ様です 🙂
先日から進めていた「ビジター招待アシスタント」プロジェクトですが、 このたび β 版（試験版）の完成に至りました。
🎯 このプロジェクトの目的
毎週、複数のメンバーが異なる方法でビジター様を招待しているため、 招待案内にバラつきが生まれています。
この課題を解決するため、
誰が使ってもブレない
統一された招待アプリ
を作成しました。
✨ このアプリでできること
✅
3 つの項目を入力するだけ
お名前、参加予定日、業種
✅
統一された案内テキストが自動生成
丁寧で一貫性のある案内が完成
✅
スケジュール・Zoom 情報・プレゼン案が全て入ったガイド
ビジター様が安心して準備できる
✅
AI がプレゼン案を自動提案
業種別のプレゼン構成案を提案
✅
メンバー表 PDF をダウンロード可能
ビジター様が事前に人間関係を把握
