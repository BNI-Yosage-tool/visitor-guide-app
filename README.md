# 🎯 BNI よさげチャプター ビジター招待アシスタント

メンバーがビジター様を招待する際に、**毎回誰もが同じ方法で、ブレずに招待できる** Web アプリケーションです。

## ✨ 主な機能

- **シンプル入力**：3つの項目を入力するだけ
- **自動生成**：統一されたテキスト、当日ガイド、メンバー表を自動生成
- **AI提案**：業種別のプレゼン構成案を AI が自動提案
- **スマートフォン対応**：iPhone での操作を最優先に設計

## 🚀 クイックスタート

1. `index.html` をブラウザで開く
2. 3つの情報を入力（ビジター名、参加日、業種）
3. 案内テキスト・ガイド・メンバー表を生成
4. ビジター様へ送信

## 📁 フォルダ構成
```
visitor-guide-app/
├── README.md
├── index.html
├── members/
│   └── メンバー表.pdf
└── docs/
    ├── 1_project_announcement.html
    ├── 2_background_explanation.html
    └── background.md
```

## 📋 メンバー表について

メンバー表は `members/メンバー表.pdf` に保存されています。

**更新方法**：
1. 最新のメンバー表を PDF で保存
2. `members/メンバー表.pdf` を新しいファイルで置き換え
3. GitHub に commit & push

アプリは自動的に最新版を参照します。

## 🎨 デザイン特徴

- **カラー**：BNI 公式カラー（赤 #E83C3C）
- **フォント**：システムフォント（高速読み込み）
- **レイアウト**：モバイルファースト設計

## 🔧 技術スタック

- **フロントエンド**：HTML5 + CSS3 + Vanilla JavaScript
- **AI統合**：Claude API（プレゼン案自動生成）
- **ホスティング**：GitHub Pages（予定）

## 📝 利用方法

### ビジター招待の流れ

1. **ステップ1**：「ビジター様を招待」を選択
2. **ステップ2**：以下を入力
   - ビジター様のお名前
   - 参加予定日
   - 業種
3. **ステップ3**：以下を送信
   - テキスト（LINE/メッセンジャー）
   - 当日ガイド HTML（メール/LINE）
   - メンバー表 PDF
